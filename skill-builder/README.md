# skill-builder

Builds and deploys skills from analysis reports generated by skill-analyzer.

## Build

```bash
chmod +x build.sh
./build.sh
```

## Usage

```bash
# Build from analysis report
./skill-builder build analysis.json

# Build from stdin
cat analysis.json | ./skill-builder build -

# Skip deployment
./skill-builder build analysis.json --skip-deploy

# Skip backup
./skill-builder build analysis.json --skip-backup

# Verbose mode
./skill-builder build analysis.json -v
```

## Output

JSON format to stdout containing:
- Build results
- Validation checks
- Deployment status
- Backup information

## Requirements

- Go 1.21+
- git command available
- Target language toolchain (Go/Rust) for building skills
